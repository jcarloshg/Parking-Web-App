#### **_Objetivo_**

El objetivo de esta prueba técnica es evaluar tus habilidades en desarrollo como Full Stack

#### **_Descripción General_**

Deberás desarrollar un sistema web para gestionar un estacionamiento público con los roles administrador, cajero y supervisor, que te permita gestionar las siguientes acciones:

- Registrar entradas y salidas de vehículos
- Calcular tarifas automáticamente
- Gestionar cajones disponibles
- Generar reportes de ingresos
- Administrar usuarios y roles

#### **_Stack obligatorio_**

- Backend: Laravel v10 o superior
- Frontend: Vue.js
- Base de datos: MySQL
- Control de versiones: Git (GitHub o Bitbucket)

#### **_Tablas esperadas base de datos_**

Tablas base, es necesario agregar mas.

parking_spaces (Gestion de cajones)

- id
- number
- type (general, discapacitado, eléctrico)
- status (disponible, ocupado, fuera_servicio)

tickets (Registro de vehiculos)

- id
- plate_number
- vehicle_type (auto, moto, camioneta)
- entry_time
- exit_time (nullable)
- parking_space_id
- status (activo, finalizado)

payments (Pago y salida)

- id
- ticket_id
- total
- payment_method (efectivo, tarjeta)
- paid_at

**Condiciones de cobro de estacionamiento:**

- Se cobra por hora completa
- Tolerancia: 10 minutos gratis
- Después de 24h aplicar tarifa diaria

#### **_Roles y usuarios_**

- Registrar entrada (Admin, cajero, supervisor)
- Registrar salida (Admin, cajero, supervisor)
- Ver reportes (Admin, supervisor)
- Gestionar usuarios (Admin)

#### **_Pantallas requeridas_**

- Dashboard
  - Cajones disponibles
  - Ingresos del día
  - Tickets activos
- Registro de entrada
  - Placa
  - Tipo de vehículo
- Registro de salida
  - Buscar ticket
  - Calcular total
  - Botón pagar
- Reportes
  - Tablas, gráficas

#### **_Puntos a considerar_**

- Paginación
- Cache (opcional)
- Autenticación con Sanctum o JWT
- Validación de inputs
- Protección CSRF

#### **_Reportes esperados_**

#### **_Diario_**

- Total ingresos
- Tickets atendidos
- Promedio por ticket

#### **_Mensual_**

- Ingresos por día
- Hora pico
- Tipo de vehículo más frecuente
